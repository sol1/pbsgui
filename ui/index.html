Metrics (Prometheus)

Expose backup metrics for Prometheus. Off by default. The metrics carry no secrets (no repositories, tokens, keys, or server hostnames), only job and database names.

Prometheus scrapes http://bind:port/metrics. Keep the bind on 127.0.0.1 unless Prometheus is on another host, and open the firewall only if you do.

Writes pbsgui.prom here for the exporter's textfile collector to pick up.
